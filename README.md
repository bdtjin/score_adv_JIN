## 심화과제: 성적 분석 프로그램
- **Collection** 활용하여 데이터 처리하기
- **파일 입출력**을 통해 데이터를 불러오고 저장하기
- 객체 지향을 활용한 전체 구조 생각하기

---

### 1. 클래스 구성: Score와 StudentScore
- **`Score (부모)클래스`**
  - 속성(점수) 정의 필요
  - 출력메서드: `(점수: 90)`

- **`StudentScore (자식)클래스`**
   - 'Score'를 상속받아서 구체적인 정보 출력 (점수 + 이름)
   - showInfo 메서드를 @override 사용해서 속성 재정의
   - 출력메서드: `이름: 홍길동, 점수: 90점`

#### [1번-트러블슈팅]
- **문제상황**
상속받은 클래스에서 정보를 출력할 때, 부모의 기능인 점수 출려과 나의 기능인 이름 출력을 결합하는 방법에 대한 구조가 잡히지 않았다.
- **분석**
`main` 함수나 자식 클래스에서 `print`문을 한번 더 쓰게되어 부모 클래스 로직을 중복해서 작성하는 비효율적임이 발생했다.
- **해결과정**
  - `Super` 키워드 활용
`super.showInfo()`를 호출하여 부모 클래스에 이미 구현된 점수 출력을 그대로 가져와서 실행했다.

---

### 2. 초기 데이터 로딩: 파일로부터 학생 점수 데이터 읽어오기

- 라이브러리의 File 클래스를 사용
- students.txt 파일에서 학생들 점수 목록을 받기
- 한 줄로 저장된 데이터를 각 줄로 쪼개서 `StudentScore` 객체로 변환 후 `List<StudentScore>`에 저장

#### [2번-트러블슈팅]
- **문제상황**
텍스트 파일에 저장된 문자열 데이터를 읽어오는 것까지는 이해가 되었으나, 이후에 코드에서 다룰 수 있는 객체로 변환해서 리스트로 묶어서 관리해야 되는지 구조 설계를 하는 과정에서 어려움을 겪었다/.
- **분석**
  - 인스턴스 개념 혼동
클래스와 객체의 차이를 명확히 구분하지 못 해서 텍스트 데이터를 바로 리스트에 넣으려 하거나 객체 생성 코드를 어디에 넣어야 작동이 잘 되는지에 대한 혼란이 있었다..
- **해결과정**
리스트를 초기화하는 변수 선언을 하고, 객체를 생성했다. 이후에 초기화한 studentList에 순차적으로 반복문이 적용될 수 있도록 하며 모든 객체에 접근할 수 있도록 했다. 

---

### 3. 사용자로부터 입력 받아 학생 점수 확인 기능
- 프로그램 시작 시 출력 메시지: `"어떤 학생의 점수를 확인하시겠습니까?"`
- 사용자로부터 이름 정보를 입력 받기 (김철수, 홍길동 해당 학생 점수 출력)
  - `StudentScore 클래스`의 `showInfo()` 메서드를 통해 출력
- 다른 값 입력 시, 메시지 출력 후 다시 입력받기
  - 출력 메시지: `"잘못된 학생 이름을 입력하셨습니다. 다시 입력해주세요."`
- 이 동작은 올바른 값이 입력될 때까지 반복

#### [3번-트러블슈팅]
- **문제상황**
3번 문제에서는 정말 많은 어려움이 있었다... 
사용자의 입력값이 리스트에 있는지 확인해야 되는데, for-in 문이 내부적으로 어떻게 리스트를 도는지 머릿속으로 그려지지가 않았다. 또한 stdin을 통해 글자를 받는 원리와 리스트에서 찾았는지 못 찾았는지 판단하는 기준을 잡기 어려웠던 것 같다.
- **분석**
for(var student in studentList)가 실행될 때, student 변수가 매 바퀴마다 리스트의 다음 요소로 교체된다는 사실을 제대로 이해 못 했었다. 반복문에서 입력값을 못 찾았다는 에러가 생기면 아직 검사하지 않은 뒤쪽 데이터는 무시하게 되는 점과 이를 해결하기 위해 검사 결과에 끝까지 참인지 거짓인지 확인해줄 수 있는 변수(bool)가 필요하다는 것을 몰랐다.
- **해결과정**
사용자 입력을 문자열로 받아서 리스트의 처음부터 끝까지 순차적으로 비교하는 구조를 만들었다. 특히 bool을 활용해서 리스트 내 값을 탐색하여 성공 여부를 정확히 판별하고, 예외 메시지를 출력하는 로직을 만들었다.

### 4. 결과 저장: 프로그램 종료 시에 결과 파일에 기록
- 실행 조건: 3번에서 사용자가 올바른 값을 입력한 경우에 해당 학생의 이름과 점수를 저장
   - 저장 형태: `이름: 홍길동, 점수: 90`
   - 종료: 저장 완료 후 `"저장이 완료되었습니다."` 메시지를 콘솔에 출력하기

#### [4번-트러블슈팅]
- **문제상황**
사용자가 올바른 값을 입력할 때까지 계속 질문해야 되는데 어디서부터 어디까지를 while로 감싸야 하는지 범위 설정이 막막했다. 또한 기존 3번 문제 풀면서 for문에서 반복을 멈추기 위해 break을 썼는데 프로그램이 종료되지 않고 계속 질문이 반복되는 현상이 발생했었다.

- **분석**
코드에 이미 for문이 들어있는 상태에서 while로 감싼 구조로 반복문이 반복이 되고 있었다. break은 자신을 감싸고 있는 가장 가까운 반복문 하나만 반복을 멈추는 기능으로 for문 탈출하고 바깥의 while문은 계속 돌아가고 있었다.

- **해결과정**
기존 break에서 반복문 탈출이 아니라 main 함수가 끝나면 함수 종료 명령어인 return을 통해 프로그램 자체가 종료되는 확실한 방법을 사용했다.
